# vuejs-essential

> 使用 Vue.js 开发的一个类似 [VuejsCaff 社区](https://vuejscaff.com/topics) 部分的网站

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
